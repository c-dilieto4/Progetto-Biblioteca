\doxysubsection{Detailed Description}
Implementazione concreta del meccanismo di persistenza su file. 

Questa classe gestisce l\textquotesingle{}interazione diretta con il file system locale per garantire il salvataggio persistente e il ricaricamento dello stato dell\textquotesingle{}archivio (Libri, Utenti, Prestiti), in conformità con il requisito di Interfaccia Software Soddisfa le esigenze di un\textquotesingle{}applicazione stand-\/alone descritte nella sezione "{}\+Prospettive del prodotto"{}.

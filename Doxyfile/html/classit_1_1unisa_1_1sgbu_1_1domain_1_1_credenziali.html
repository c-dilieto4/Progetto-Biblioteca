Detailed Description

Rappresenta le credenziali di accesso al sistema.

Questa classe incapsula le informazioni di autenticazione necessarie per il login. È destinata all'uso esclusivo dell'attore Bibliotecario, l'unico autorizzato ad accedere all'interfaccia grafica per eseguire operazioni di gestione.

Implementa la struttura dati richiesta per soddisfare il requisito di sicurezza sull'Autenticazione.

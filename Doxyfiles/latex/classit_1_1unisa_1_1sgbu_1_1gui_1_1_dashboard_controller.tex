\doxysubsection{Detailed Description}
Controller per la Dashboard principale (GUI). 

Gestisce l\textquotesingle{}interazione tra l\textquotesingle{}utente e il sistema attraverso l\textquotesingle{}interfaccia grafica. Si occupa di popolare le tabelle, gestire gli eventi dei pulsanti e coordinare i flussi di visualizzazione dati.

Metodo di inizializzazione automatica Java\+FX. 

Viene invocato automaticamente dopo il caricamento del file FXML. Configura le proprietà visuali delle tabelle e le logiche delle celle personalizzate.

\begin{DoxyPostcond}{Postcondition}

\begin{DoxyItemize}
\item Le policy di ridimensionamento delle colonne sono impostate su CONSTRAINED\+\_\+\+RESIZE.
\item La colonna "{}\+Ritardo"{} è configurata per mostrare i relativi stati.
\item Lo stile della colonna "{}\+Ritardo"{} cambia colore (Rosso/\+Verde) in base allo stato. 
\end{DoxyItemize}
\end{DoxyPostcond}

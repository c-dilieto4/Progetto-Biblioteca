Detailed Description

Classe Boundary che gestisce l'interfaccia utente (View).

Questa classe orchestra la visualizzazione delle finestre (Login, Dashboard), la creazione delle finestre di dialogo (Input/Alert) e l'aggiornamento dei componenti grafici delegando al DashboardController.

Carica e mostra la Dashboard principale (dopo il login).

Precondition
  • Il file "DashboardView.fxml" deve essere presente nelle risorse.
Postcondition
  • Il file FXML viene caricato.
  • Il DashboardController viene inizializzato e collegato.
  • Le tabelle vengono popolate con i dati attuali.
  • La scena viene impostata sullo stage primario e mostrata.

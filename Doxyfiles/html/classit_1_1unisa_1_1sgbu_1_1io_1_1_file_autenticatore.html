Detailed Description

Implementazione del meccanismo di autenticazione basato su file.

Questa classe verifica le credenziali del Bibliotecario leggendole da un supporto di memorizzazione persistente. Soddisfa il requisito Non Funzionale di Sicurezza, garantendo che l'accesso alle funzionalità di gestione sia limitato al solo personale autorizzato (Attore: Bibliotecario).

Verifica la validità delle credenziali fornite.

Implementa il controllo di sicurezza. Confronta l'input fornito con le credenziali memorizzate su file. Il successo di questa operazione è la Pre-condizione necessaria per l'esecuzione di tutti i casi d'uso gestionali.

Parameters
[in]userIl nome utente inserito dall'operatore.
[in]passLa password inserita dall'operatore.
Returns
true se le credenziali corrispondono a quelle autorizzate, false altrimenti.
Precondition
  • Il file delle credenziali deve esistere ed essere leggibile.
Postcondition
  • Se restituisce true, il Bibliotecario è considerato "Autenticato" nel sistema.
